    Blaseball Tools

    These are a few blaseball clomputational tools published by Ch4zm of Hellmouth, useful for exploring blaseball games and teams:

    • interesting-blaseball-games - a command-line tool for finding interesting blaseball games

    • blaseball-streak-finder - a command-line tool for finding blaseball winning and losing streaks
